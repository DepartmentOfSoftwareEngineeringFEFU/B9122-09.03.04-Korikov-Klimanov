# Защищённый мессенджер «Эгида»

<img src="assets/app_logo.png" alt="Описание" width="300" height="300">

## Описание проекта

**«Эгида»** - программное средство для безопасного обмена сообщениями и медиафайлами со сквозным шифрованием данных (End-to-End Encryption, E2EE).
Система позволяет пользователям вести конфиденциальную переписку в реальном времени, обеспечивая защиту передаваемой информации от перехвата и несанкционированного доступа на всех этапах передачи.

Одной из ключевых особенностей системы является многоуровневая модель безопасности: сквозное шифрование сообщений на клиентах, двухфакторная аутентификация (TOTP), биометрическая защита входа в приложение (ПИН-код, отпечаток пальца и Face ID), а также поддержка секретных чатов с исчезающими сообщениями по таймеру. Серверная часть отвечает за производительную маршрутизацию сообщений через WebSocket и хранение зашифрованных данных.
